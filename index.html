Show print settings
  1. Open the browser print dialog with Print card.
  2. Set Paper size to A6.
  3. Set Orientation to Landscape.
  4. Set Scale to 100% or Default, not "Fit to page".
  5. Set Margins to None or Default.
  6. Turn Headers and footers off.
  7. For double-sided printing, use Duplex with flip on short edge.

If the print is too large, the paper size is usually still set to A4 or Letter in the print dialog or printer driver.
